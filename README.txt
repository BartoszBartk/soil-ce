This is the repository for code and data used for a discrete choice experiment into public preferences for soil-based ecosystem services. The study was funded by the German Federal Ministry of Education and Research (BMBF) in the framework of the funding measure “Soil as a Sustainable Resource for the Bioeconomy – BonaRes”, project “BonaRes (Module B): BonaRes Centre for Soil Research, subproject A” (grant 031B0511A).
